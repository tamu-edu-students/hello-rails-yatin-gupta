# README

This project is an introduction to Rails !

* Ruby version - 2.7.2

Install Dependencies
```shell
bundle install
```

Running migrations
```
rake db:migrate
```

Heroku commands
```
heroku run rails db:migrate
heroku stack:set heroku-20 --app hello-world-rails-yatin // since rails 2.7.2 is not supported on heroku 22
```
